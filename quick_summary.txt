📝 IMPLEMENTATION SUMMARY
─────────────────────────────────────────────────────────────────────────────

Desugarer Strategy:
  • Static→Static: List.of() → CollectionBackport.listOf()
    (Same descriptor, new owner/name, isInterface=false)

  • Instance→Static: stream.takeWhile(p) → StreamBackport.takeWhile(stream, p)
    (Receiver prepended, INVOKEVIRTUAL→INVOKESTATIC)

  • No stack manipulation needed (receiver already below args)

Architecture:
  ClassDesugarer (ASM ClassVisitor)
    ├─ Downgrades version 53 → 52
    ├─ Strips ACC_PRIVATE from interface methods
    └─ Delegates method bodies to MethodDesugarer

  MethodDesugarer (ASM MethodVisitor)
    ├─ Intercepts visitMethodInsn() calls
    ├─ Matches ~150+ API call patterns
    ├─ Rewrites to backport calls
    └─ Handles string concatenation rewriting

  j9compat Runtime Library
    └─ 22 backport classes (all Java 8 compatible)
